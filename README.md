## Project setup
```
npm install
npm run serve
npm run build
npm run lint
```

## Заметки

В коде используется мой личный шаблон, 
поэтому некоторые scss-переменные, scss-классы, компонетнты и т.п. могут быть неиспользуемыми.  
  
Современные минификаторы и браузеры оптимизируют код достаточно хорошо.
