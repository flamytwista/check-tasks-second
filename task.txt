Models
======
note
    - id
    - name (String)
task
    - id
    - name (String)
    - checked (Boolean)
    - note_id (Number)


Приложение состоит всего из 2х страниц.

Страница - Список заметок.
=========================
кнопка "создать"
Все заметки.

Страница - создать/изменить заметку.
+=========================
Поле "имя" (текст)

    [список карточек task-ов]
    Кнопка "добавить task"

Кнопка"Добавить/Изменить"
Кнопка "отменить редактирование" (подтверждение)
Кнопка "удалить" (подтверждение)


-=========================




Карточка заметки
+================

name,
заголовки нескольких первых task
Кнопка "изменить"
кнопка "удалить" (необходимо подтверждение)
-================

Карточка (элемент списка) task
+================
Чекбокс редакритуемый
Поле "имя" (текст)
Кнопка "Удалить" (про подтверждение не написано)

-================




Прочие важности
===============
Для подтверждений модалки должны быть самопальные
После перезагрузки страницы состояние списка заметок должно сохраняться.
Кнопки "Undo" и "Redo" при редактировании заметки (горячие клавиши тоже должны работать). Думаю во vuex хранить изменаемую заметку в отдельном поле и следить за её мутациями.
